(论文总结)EGNet:Edge Guidance Network for Salient Object Detection

Posted on 2020-04-30 Edited on 2020-05-12 In 论文总结 Views:

ICCV 2019, 论文链接

本文试图解决一个问题：主流的基于U-net的显著性检测网络虽然获得了较好的结果，但是由于先前的网络过多的考虑了像素级因素，缺少结构性信息，导致检测结果，尤其是边界不够准确。因此本文也引入了额外的网络结构针对边缘进行检测，以提高预测结果。模型结构如下图所示。

本文与主流显著性检测网络一样采用了基于U-net的结构。特征提取方面，本文采用了VGG16，不过特别的是模型中仿照DSS对最后一层添加了一条side path，从而获得了六层特征图，但考虑到第一层特征图抽象程度过低，文中依然只使用了后五层特征图。

除特征提取网络外，模型可分为三个部分：PSFEM(progressive salient object features extraction module)； NLSEM(non-local salient edge features extraction module)；O2OGM(one-to-one
guidance module)。

NLSEM其实就是边缘检测分支，与其他采用了这一策略的模型在思想上相似；PSFEM是U-net结构在Top-Down部分的变体，本质还在于融合、优化多尺度特征。其在每个shortcut上添加了三个卷积层以提高模型鲁棒性；最终通过O2OGM将边缘信息融入特征图，其输出经过多次卷积得到最终的推断结果。整体结构和思路很清晰。

损失函数上，模型对NLSEM模块的输出添加了边缘loss，对多个尺度显著性推断结果均添加了交叉熵loss（即deep supervision），而不仅仅对最终的输出添加loss。

模型在ECSSD、PASCAL-S、DUT-O、HKU-IS、SOD、DUTS-TE数据集上进行了测试。
